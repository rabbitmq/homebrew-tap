# RabbitMQ Homebrew Tap

This [Homebrew](https://brew.sh/) tap contains formulas for a number of RabbitMQ-related tools.

## Installation

```shell
brew tap rabbitmq/tap
```

## Formulas

 * `rabbitmqadmin`: modern RabbitMQ CLI based on the HTTP API
 * `fshc`: file and socket handle counter
